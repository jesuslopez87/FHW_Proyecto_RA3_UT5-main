## 1. Identificación

- **Versión:** 23.1
- **Edición o sabor:** Debian base
- **Arquitectura:** 64 bits
- **Enlace oficial de descarga:** [Enlace](https://antixlinux.com/download/)

## 2. Requisitos y características
- **RAM mínima indicada por la fuente:** 256MB 
- **Espacio en disco mínimo:** 5GB
- **Tipo de entorno de escritorio:** Gestor de ventanas IceWM
- **Tamaño aproximado de la ISO:** 2,1 GB

## 3. Motivos de selección

Es una de las mejores distribuciones para hardware heredado porque prescinde de systemd. Al usar IceWM, ofrece un entorno visual basico y completo con un consumo de RAM minimo (menos de 200 MB en reposo), dejando casi todo el GB de memoria libre para las aplicaciones del usuario.

## 4. Posibles riesgos o dudas

La interfaz puede resultar un poco anticuada o poco intuitiva para alumnos acostumbrados a Windows 10/11. Además, al no usar systemd, algunos comandos de administración de servicios en clase serán diferentes.

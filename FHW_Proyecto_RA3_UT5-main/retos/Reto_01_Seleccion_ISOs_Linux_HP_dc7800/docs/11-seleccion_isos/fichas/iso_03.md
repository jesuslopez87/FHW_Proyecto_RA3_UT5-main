# ISO 03

## 1. Identificación
- **Nombre de la distribución:** Tiny Core Linux
- **Versión:** 10.0.2
- **Edición o sabor:** Debian base
- **Arquitectura:** 64 bits
- **Enlace oficial de descarga:** [Enlace](http://tinycorelinux.net/downloads.html)

## 2. Requisitos y características
- **RAM mínima indicada por la fuente:**  46 MB
- **Espacio en disco mínimo:** ~50 MB
- **Tipo de entorno de escritorio:** Gestor de ventanas FLWM  
- **Tamaño aproximado de la ISO:** 50 MB

## 3. Motivos de selección

Es la expresión mínima de un sistema Linux. Si el equipo tuviera componentes dañados o la RAM fallara limitando aún más la capacidad, Tiny Core es capaz de arrancar en casi cualquier chatarra electrónica. Es un "salvavidas" absoluto.

## 4. Posibles riesgos o dudas

Viene completamente "pelado". No tiene navegador web completo, ni suite ofimática preinstalada. El usuario tendria que montar y descargar paquete a paquete mediante su gestor appbrowser, lo cual alarga mucho los tiempos de la práctica.
